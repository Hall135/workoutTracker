Activating HTTP Live Server Using Python:
    CMD: python -m http.server 8000
    Google: http://localhost:8000

Activating HTTPS Live Server Using Python:
    CMD: python -m http.server 5500 --bind 0.0.0.0 --directory "Served Directory Path" --ssl-certfile "cert.pem Path"  --ssl-keyfile "key.pem Path"
